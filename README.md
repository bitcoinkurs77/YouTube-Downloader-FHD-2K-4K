## üî• Key Features

- **Original quality without re-compression** - video and audio are taken directly from YouTube streams (VP9 / AV1 / AAC, etc.)
- Support for **FHD (1080p), 2K (1440p), 4K (2160p),** and lower - if the video is uploaded in this resolution
- Very high download speed (multi-threaded download)
- Works with almost **any** YouTube video (regular videos, Shorts, stream archives, etc.)
- The simplest interface - literally 1-2 windows
- Does not require manual installation of ffmpeg / yt-dlp (everything is built into the .exe version)
- Minimalistic and clear design
